# Rasul — GTM Engineer Website

Personal portfolio site for an AI GTM Engineer. Live at **[rasul-site.vercel.app](https://rasul-site.vercel.app)**.

## What It Is

Single-page HTML site with:
- Retro/Darth Vader/Illuminati theme with blackhole + spinning spaceship Three.js background
- Full professional profile: proof, case studies, testimonials, tech stack, process
- Interactive "GTM Anomaly Destroyer" space shooter game
- All contact info baked in

## Stack

- Single `index.html` — no build step, no dependencies
- Three.js (CDN) for 3D background
- Inline CSS + vanilla JS
- Hosted on Vercel

## Deploy

```bash
vercel --prod
```
